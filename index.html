Data Processing & Compliance

Our platform follows Google Play Developer Policy and collects data only for essential transportation and safety services.

  • Data is collected only with user consent
  • Aadhaar & Driving License used only for identity verification
  • Documents are stored securely in Firebase Firestore
  • Only authorized admin can access verification data
  • Redis is used only for temporary live tracking cache (not permanent storage)
  • Location data is used only during active trip sessions

Data Retention Policy

We retain user data only for as long as necessary to provide services, ensure security, and comply with legal obligations.

  • Verification documents: Stored until account deletion request
  • Location data: Temporary and session-based
  • Trip history: Stored for service improvement
  • Users can request full deletion anytime

Identity Verification Documents

Aadhaar Card and Driving License are collected strictly for identity verification, fraud prevention, and ensuring safety of passengers and drivers.

These documents are securely stored in Firebase Firestore with restricted admin-only access. We do not sell, share, or misuse any verification data.
  • Encrypted storage
  • Admin-only verification access
  • User consent required before upload
  • Can be deleted on user request
